Welcome to Suburbia

Welcome to Suburbia is one of my main game projects and a strong example of the kind of work I want to keep building. The game mixes multiplayer interaction, procedural neighborhood ideas, humor, horror, and environmental storytelling into a strange world that feels playful and unsettling at the same time.

Role: Game developer, system designer, worldbuilder, and creative direction.
Tools: Godot, multiplayer systems, gameplay scripting, procedural ideas, 3D workflows.
Notes: This project reflects a lot of my interests in games: atmosphere, weird systems, player discovery, multiplayer interaction, and taking a familiar setting like suburbia and turning it into something increasingly wrong and memorable.

Cheyenne Mountain Zoo Redesign

This project is a UX case study centered on improving the zoo’s digital experience. The work focused on usability, structure, visual design, and a more polished ticket-purchase flow while keeping the site easier to understand and navigate.

Role: UX designer, UI designer, researcher, and case study builder.
Tools: Figma, wireframing, visual design, prototype work, usability thinking.
Notes: The project involved identifying pain points in the existing experience, improving navigation and clarity, and creating a cleaner, more intentional interface with a stronger sense of brand and usability.

LearnMaster

LearnMaster is a collaborative web application designed to make studying more organized and flexible. It gives users tools to build their own study sets, review flashcards, and take quizzes in a more customized learning environment.

Role: Contributor on full-stack web development and project implementation with Colin O'Connor, Kyle Okura, Adarsh Charugundla, and Matthew Egyed.
Tools: HTML, CSS, JavaScript, Node.js, Express.js, PostgreSQL, Handlebars, bcrypt, sessions, Docker.
Notes: This project shows my experience with collaborative web development, authentication, backend systems, database work, and building a practical user-focused application from concept to deployment.

RGB Matrix Room Counter

Click to expand + flip

RGB Matrix Room Counter

This project combines physical computing, display output, enclosure thinking, and interaction design. The goal was to create a system that could sense room presence and communicate that count in a clear, engaging way through an RGB matrix.

Role: Interactive systems builder, prototyper, and enclosure designer.
Tools: Arduino-style prototyping, sensors, matrix display work, enclosure design, fabrication.
Notes: This project reflects my interest in connecting code, electronics, fabrication, and visual output into one cohesive build. It shows how I approach both technical functionality and the physical presentation of a project.
